# ❤️ Rifa Solidaria

Aplicación web desarrollada para administrar una rifa solidaria destinada a recaudar fondos para una cirugía.

## Características

- 🎟️ Rifa con números del 00 al 99.
- ☁️ Reservas en tiempo real utilizando Firebase Firestore.
- 📱 Diseño adaptable para celular, tablet y computador.
- 🔴 Visualización de números disponibles y reservados.
- 👤 Registro de nombre y teléfono del participante.
- 🎁 Información del premio.
- 📅 Fecha del sorteo.
- ❤️ Mensaje de agradecimiento a los participantes.

## Tecnologías

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase
- Firestore
- Netlify

## Objetivo

Brindar una plataforma sencilla, moderna y segura para que cualquier persona pueda reservar un número y apoyar una causa solidaria destinada a recaudar fondos para una cirugía.
